How It Work?

Examine App Binaries

AppCensus does static analysis of what sensitive data an app’s code might access. We go beyond existing tools by determining whether apps really access the data in practice.

Instrument Operating

AppCensus does dynamic analysis to monitor what the apps actually do, including accessing personal data through unofficial non-API mechanisms like filesystem inspection.

Human or Automated

AppCensus operates the app to observe its real behaviors. Humans can make sure specific complex interactions are tested, while automation lets us test app behaviors at scale.
